System Requirements

As a cloud-based, web application, SOTI Snap requires a browser and internet connection. Your browser must meet the following requirements to access the SOTI Snap console. For best performance, always use SOTI Snap with the latest version of a supported browser.

Supported Internet Browsers

Internet Browser Minimum Version Recommended Level
Google Chrome 49 Latest Version
Mozilla Firefox 52 Latest Version
Apple Safari 10 Latest Version

Supported Devices

Apps built in SOTI Snap are accessed through the SOTI Snap agent, available for Android and iOS devices.

Operating System Minimum Supported Version
Android 4.4

Download from the Google Play store
iOS 9

Download from the Apple App Store
