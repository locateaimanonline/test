Possible Import Issues for Shareable Links

If you run into a problem while importing shareable links, one of the following may be the cause.

Error Message Possible Cause Fix
This link no longer exists The creator of the shareable link has deleted the link. Ask the creator of the shareable link to generate a new link.
This link has expired The expiration date set on the shareable link has passed and the link is no longer valid. Ask the creator of the shareable link to update the expiry date of the link and update the shareable link or to generate a new link.
This link isn't compatible with your SOTI Snap version The shareable link was exported from a newer version of SOTI Snap than your own SOTI Snap console.
  • Update your SOTI Snap environment.
  • Contact SOTI Support to fix the issue
This link has reached its import limit The link has met its import limit. Ask the creator of the shareable link to increase the import limit and update the shareable link or to generate a new link.
