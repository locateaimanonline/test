Events and Actions

Increase the functionality of your apps with Events and Actions. In SOTI Snap you can configure app events such as clicking a button or leaving a text box to trigger certain actions. For example, if your app has a form, you can add a Submit button that, when clicked, automatically submits and synchronizes data from the form to the SOTI Snap server and then clears the form to prepare it for the next entry.

SOTI Snap supports several event types, each with multiple actions you can use.

Note: Not all widgets support events and actions. In the Canvas, you can see which widgets allow Events by checking the widget configuration panel on the right. The Events tab will be completely disabled on widgets which do not support events. Furthermore, even if a widget does support events, it likely does not support all events.

Events

Event Description
OnBlur Is triggered when the app user leaves a previously active input field such as a text box.
OnChange Is triggered when the app user changes the selected option to another option, such as in a dropdown list or between radio buttons.
OnClick Is triggered when a button is clicked.
OnLoad Is triggered when the app page that contains the widget is loaded.
OnRead Is triggered when the app reads an NFC tag.
OnScan Is triggered when the app scans a barcode.
On Select / unselect Is triggered when the app user selects one or more options from a widget.

Actions

Action Description
Submit Automatically submits data entered into a form.
Note: The Submit action is only available on pages nested under Form screens.
Sync Automatically synchronizes data between the device and the SOTI Snap server, depending on the app's data sync settings: 1 way sync only uploads data from the device to the server, while 2 way sync synchronizes data between the device and the server.
Note: The Sync action is only available on pages nested under Form screens.
Navigate Sends app user to a specified page or location within the app.
Navigate Back Sends app user back to their previous location within the app.
Validate Page Checks that the content entered on the page conforms to any set validity standards.
Reset Clears pages of entered data.
Run JavaScript Executes custom JavaScript code when the event is triggered. Paste in your custom JavaScript code or click Script Editor to open a larger editing field where you can type in your code.
Run Blockly Executes custom code developed using Blockly when the event is triggered. Blockly is a visual programming interface that uses colorful blocks to approximate coding concepts in an easy to understand manner and requires no programming experience. It allows SOTI Snap administrators who are not familiar with JavaScript to develop their own custom actions.

See Run Blockly for an example.

Note: Learn more about Blockly in general at the Google Blockly Reference Guide.
Open Web Browser Launches the device's default web browser and navigates to the specified URL.
Open Calendar Launches the device's default calendar app.
Send SMS Sends an SMS to the specified phone number.
Send Email Sends an email to the specified email address.
Launch Map Open the device's default map app to the specified location. Enter a URL for Google Maps or Bing Maps in the Location URL field.
Call Calls the specified phone number.
Launch SOTI Surf Opens the SOTI Surf app.
Launch SOTI Hub Opens the SOTI Hub app.
Invoke Data Export Initiates any data transfers as specified in the app's Data Export Connections settings.
Invoke Data Import Initiates any data transfers as specified in the app's Data Import Connections settings.
Map Columns Populates the field of a linked widget (such as a textbox) with content an app user selects from an external data widget. You can map different columns in the external source to different widgets.
Map Read Value Pulls data read from an NFC widget and inserts it into the specified fields within the app.
Map Scanned Value Pulls data scanned from a barcode widget and inserts it into the specified fields within the app.
Exit App Closes the app and returns to the app catalog.
