Creating Report Templates

About this task

To design a custom template for a report:

Procedure

  1. Open the Apps view and beside the published app whose records you want to export, select More > Templates.
  2. Select New Template
  3. Enter a name for the template.
  4. Select the form where the app fields you want to include in the report appear.
    You cannot include app fields from multiple forms.
  5. Move the app fields you want to include in the report template from App Fields to Include in PDF.
  6. Turn on Don't show empty fields in PDF if you want to hide fields that don't contain any data.
  7. Select Create.

Results

Templates appear in the Templates sub view of the Apps view.

Tip: Once you create your first custom report template, the default SOTI Snap template that includes all app fields is no longer available. You may want to create a custom template that includes all app fields.
